# 📌 Overview

This repository contains two small **desktop games built using Python's Tkinter library**.  
The goal of the project is to practice **GUI programming and game logic**.

Each game is implemented as a **standalone Python script** that can be run independently.

---

# 🎮 Games Included

## 1️⃣ Tic Tac Toe

A classic **two-player Tic Tac Toe game** with a graphical interface.

### Features

- Interactive **3×3 game grid**
- Alternating player turns
- Win detection
- Draw detection
- Restart option

### Concepts Used

- Tkinter buttons and grid layout
- Game state management
- Win condition checking

---

## 2️⃣ Word Guesser

A **word guessing game** where the player tries to guess the hidden word letter by letter.

### Features

- Random word selection
- Letter-by-letter guessing
- Game progress display
- Win/lose conditions

### Concepts Used

- Tkinter GUI components
- String manipulation
- Random module
- Event handling

# ⚙ Installation

Clone the repository

```
git clone https://github.com/no-usefun/Python-mini-games.git
```

Navigate to the project directory

```
cd Python-mini-games
```

---

# ▶ Running the Games

Run any game using Python.

### Run Tic Tac Toe

```
python tic_tac_toe.py
```

### Run Word Guesser

```
python word_guesser.py
```
