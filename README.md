# 🎮 Python Mini Games Collection

This repository contains two simple desktop games built using **Python** and **Tkinter**.  
These projects were developed for learning purposes to understand GUI programming, event handling, and basic game logic in Python.

---

# 1️⃣ Jumbled Words Game

## 📌 Description

A word guessing game where a scrambled word is displayed and the user must enter the correct answer in lowercase.  
The score increases for correct answers and decreases when skipping a word.

---

## 🛠 Technologies Used

- Python 3
- Tkinter (GUI)
- Random module
- Tkinter MessageBox

---

## 🎯 Features

- Random word selection
- Score tracking system
- Skip option (with score penalty)
- Input validation
- Restart/New Game option
- Error popups for incorrect answers

---

## ▶️ How to Run

1. Ensure Python 3 is installed.
2. Save the file as `jumbled_words.py`
3. Run:

```
python jumbled_words.py
```

---

## 🎮 Game Rules

- Words must be entered in **lowercase**.
- Correct answer → +1 score.
- Skip word → -1 score (if score > 0).
- Score cannot go below zero.

---

# 2️⃣ Tic Tac Toe Game

## 📌 Description

A two-player Tic Tac Toe desktop game built using Tkinter.  
Players alternate between X and O, and the game automatically detects wins and draws.

---

## 🛠 Technologies Used

- Python 3
- Tkinter (GUI)
- Tkinter MessageBox

---

## 🎯 Features

- Two-player mode (Player 1: X, Player 2: O)
- Automatic win detection
- Draw detection
- Score tracking for both players
- Reset board option
- New Game option (resets scores)
- Exit button

---

## ▶️ How to Run

1. Ensure Python 3 is installed.
2. Save the file as `tic_tac_toe.py`
3. Run:

```
python tic_tac_toe.py
```
